# LinkedIn Post Template

Professional LinkedIn posts optimized for engagement and reach.

---

## How to Use

1. Copy the template below for your LinkedIn post
2. Replace all `{{VARIABLE}}` placeholders with your content
3. Keep the main post link-free (links go in first comment)
4. Post the first comment immediately after publishing

## LinkedIn Formatting Rules

### Character Limits

| Zone | Limit | Importance |
|------|-------|------------|
| Above fold (desktop) | 210 chars | Only visible before "See more" |
| Above fold (mobile) | 140 chars | Even more critical |
| Total post | 3,000 chars | Maximum allowed |
| Optimal engagement | 1,300-1,600 chars | Best performance |

### Text Formatting

**LinkedIn does NOT support native bold/italic.** Must use Unicode converters.

| Format | How to Create | Notes |
|--------|---------------|-------|
| Bold | Unicode converter tool | Use for emphasis only |
| Italic | Unicode converter tool | Use sparingly |
| Regular | Type normally | For most content |

**What does NOT work:**
- `**bold**` (Markdown) - shows as plain text
- `*italic*` (Markdown) - shows as plain text
- `# Headers` (Markdown) - shows as plain text
- HTML tags - stripped or shown as text

### Line Breaks

- Line breaks ARE preserved
- Use short paragraphs (1-3 sentences max)
- Single blank line between paragraphs
- White space improves readability

### Hashtags

| Rule | Recommendation |
|------|----------------|
| Quantity | 3-5 maximum |
| Placement | End of post |
| Style | #CamelCase for multi-word |
| Integration | Can weave into text naturally |

### Links

**CRITICAL**: LinkedIn algorithm penalizes external links in main post.

| Strategy | Implementation |
|----------|----------------|
| Video/resource links | Put in FIRST COMMENT |
| Main post | Keep link-free for better reach |
| URLs | Always include `https://` |

### Emojis

| Usage | Recommendation |
|-------|----------------|
| Quantity | 3-5 per post maximum |
| Type | Functional emojis only |
| Placement | Section markers, emphasis |

**Recommended emojis:**
- Insights, ideas
- Goals, targets
- Completed, confirmed
- Launch, growth
- Data, metrics

## Post Type Templates

### Lesson Learned Post

```
I made a mistake that cost me [X hours/dollars/etc].

Here's what happened:

[Brief story of the problem]

The lesson?

[Key insight in 1-2 sentences]

Now I [what you do differently].

Have you experienced something similar?

#Hashtag1 #Hashtag2 #Hashtag3
```

### How-To Post

```

Step 2: [Action]
Step 3: [Action]

The key insight most people miss:

[Critical detail that makes the difference]

#Hashtag1 #Hashtag2 #Hashtag3
```

### Opinion/Insight Post

```

[Supporting point 1]

[Supporting point 2]

[Supporting point 3]

[Your conclusion or recommendation]

#Hashtag1 #Hashtag2 #Hashtag3
```

## Quality Checklist

Before publishing:

**Content:**
- [ ] Hook grabs attention in first 140-210 chars
- [ ] 1,300-1,600 chars total (optimal range)
- [ ] Short paragraphs (1-3 sentences)
- [ ] Clear question as CTA
- [ ] No harsh language about open-source projects
- [ ] No double dashes or em dashes

**Formatting:**
- [ ] 3-5 hashtags at end
- [ ] #CamelCase for multi-word hashtags
- [ ] NO links in main post
- [ ] First comment prepared with links

**Links (First Comment):**
- [ ] All links include `https://`
- [ ] Twitter uses `twitter.com` (not `x.com`)
- [ ] Facebook uses `facebook.com` (not `fb.com`)

---

## Algorithm Notes

LinkedIn prioritizes:
- Native content over link-outs
- Dwell time (longer reading)
- Early engagement (first hour critical)
- Posts without external links

Optimize for:
- Hook that stops scrolling
- Short paragraphs for easy reading
- Question that invites comments
- Links in comments only
